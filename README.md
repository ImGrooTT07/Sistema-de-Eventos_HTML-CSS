# 📌 Sistema de Eventos

Este projeto é uma página web desenvolvida com **HTML e CSS**, utilizando **Flexbox** para organizar todos os elementos da interface.

O objetivo é simular um sistema de visualização de eventos, com layout moderno e responsivo.

---

## 🚀 Funcionalidades

- 📍 Menu de navegação no topo  
- 🎯 Banner com chamada principal  
- 📂 Filtros de categorias  
- 📅 Lista de eventos em formato de cards  
- ⭐ Destaque para evento principal  
- 📱 Layout responsivo  

---

## 🎨 Tecnologias Utilizadas

- HTML5  
- CSS3  
- Flexbox  

---

## 📐 Organização com Flexbox

O projeto utiliza Flexbox para estruturar toda a página:

- **Header:** alinhamento horizontal entre título e menu  
- **Banner:** centralização vertical e horizontal  
- **Main:** divisão entre filtros (esquerda) e eventos (direita)  
- **Filtros:** organização em coluna  
- **Eventos:** cards organizados verticalmente  
- **Footer:** conteúdo centralizado  

---

## 💡 Conceitos Aplicados

- `display: flex`
- `justify-content`
- `align-items`
- `flex-direction`
- `flex-wrap`
- `gap`

---

## 📱 Responsividade

O layout se adapta para telas menores:

- Filtros passam para o topo  
- Cards se reorganizam automaticamente  
- Conteúdo centralizado  


---

## 📷 Resultado

Interface limpa, organizada e com melhor aproveitamento de espaço, simulando um sistema real de eventos.
